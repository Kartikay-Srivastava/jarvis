# JARVIS 🤖

JARVIS is an intelligent virtual assistant designed to automate tasks. The project aims to replicate the functionality of a smart AI assistant capable of voice interaction.

---

## 🚀 Features

- 🎤 Voice recognition 
- 🌐 Web search integration
- 🖥️ System command execution
- 📅 Task automation
- 🧠 Extensible modular architecture
- 🔌 API integrations (future will be added)

---

## 🛠️ Tech Stack

- **Language:** Python , JavaScript **
- **Frameworks:** Textillate , BootStrap, Tailwind**
- **Speech Recognition:** SpeechRecognition 
- **Text-to-Speech:** pyttsx3 
- **Backend:** Python
- **Frontend :** React / HTML / CSS

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/kartikay-srivastava/jarvis.git
cd jarvis
------------
pip install -r requirements.txt
----------
python main.py  or python run.py work more fine
```
